# Legacy Workbench Route

The root `/workbench` page is intentionally passive and redirects to `/workbench-v2`.
Do not re-enable or edit the old simple/advanced workbench shells for active product
work unless a task explicitly asks for the legacy surface.

Keep `/workbench/proposal` active. The current `/workbench-v2` calculator still uses
that nested route for report preview, report editing, and report handoff flows.
