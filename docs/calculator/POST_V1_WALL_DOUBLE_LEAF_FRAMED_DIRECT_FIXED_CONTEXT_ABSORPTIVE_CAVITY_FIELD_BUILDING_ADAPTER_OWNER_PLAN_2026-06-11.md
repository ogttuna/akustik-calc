promotion. Broad source crawling is lower ROI because the selected gap is
already created by owned runtime inputs and owned Gate EO/Gate I/Gate AR
formula routes. A-weighted `Dn,A` / `DnT,A` remains a later companion
owner because the official Gate ER target list is still `R'w`, `Dn,w`,
and `DnT,w`.

Decision: select the bounded direct-fixed context absorptive cavity
field/building adapter owner. It should convert four currently
unsupported request shapes into calculated outputs:

- full context-owned absorptive cavity with `field_between_rooms`;
- partial context-owned absorptive cavity with `field_between_rooms`;
- full context-owned absorptive cavity with `building_prediction`;
- partial context-owned absorptive cavity with `building_prediction`.

`Dn,w`, and `DnT,w`, with no formula retune and no source rows imported.
