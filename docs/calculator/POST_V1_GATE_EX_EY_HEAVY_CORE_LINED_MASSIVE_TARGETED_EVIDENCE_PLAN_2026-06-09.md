the current implementation still lacks a wall-specific lined concrete or
heavy-masonry source row and lacks a named bounded wall lining rule.

Gate EX selects targeted evidence acquisition, not a broad source crawl.

a future calibration owner:

- a wall-specific lined concrete or heavy-masonry source row with

  tolerance, holdouts, and negative boundaries.

This is calculator accuracy work because it protects the route from
unsafe retuning while defining the exact evidence needed before values
can move. It is not a catalog-building task.

evidence remains insufficient, Gate EY must keep the owner rejected and
select the next bounded calculator action.
