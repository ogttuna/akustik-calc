mixed-basis calculator request more complete without weakening metric
integrity: a user can enter a custom visible heavy floating-floor stack,

broad material-editor task, or formula retune. It must not alias ISO
`Ln,w`, `DeltaLw`, `L'n,w`, or `L'nT,w` to ASTM `IIC` / `AIIC`.
