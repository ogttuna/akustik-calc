outputs, but `C` and `Ctr` were still unsupported for complete indoor
field/building door/window/facade requests. The existing Gate S
opening/leak loss and host-wall transmission-loss curve can produce

runtime values or target outputs.

shared output schema does not expose `OITC` and the runtime adapter is
not owned. Scalar STC opening ratings remain blocked because Gate AH
owns STC from a shifted transmission-loss spectrum, not from a scalar
rating alias.

Rejected parallel support work:

opening-element modeling remain useful future lanes, but they do not
outrank the immediately calculable C/Ctr spectrum companion owner.
