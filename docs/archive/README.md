# Archive Docs

This folder keeps dated historical notes that are useful for context but are not the primary source of truth.

If an archived note disagrees with a living document under `docs/calculator` or `docs/foundation`, the living document wins.

Subfolders:

- `status`
  - dated project snapshots retained for chronology
- `handoffs`
  - dated implementation-pass handoff notes
- `analysis`
  - dated investigation logs and numerical deep dives
